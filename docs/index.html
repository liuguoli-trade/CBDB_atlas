關於 CBDB Atlas

中國歷代人物傳記資料庫（CBDB） 是面向中國歷史研究的結構化傳記資料庫。CBDB Atlas 將查詢邏輯外置於 queries/，在本地以 Web 界面提供人物檢索、傳記詳情、關係可視化與 Excel 匯出等功能。

GitHub 源碼 CBDB SQLite 官方發布

主要功能

  • 人物、地名等多類型檢索與傳記詳情
  • 關係可視化（單人、家族、交游、關係探索）
  • 網頁一鍵下載並校驗官方 CBDB 資料
  • 人物詳情匯出 Excel

本地安裝與使用

  1. 克隆倉庫：git clone https://github.com/liuguoli-trade/CBDB_atlas.git
  2. 安裝依賴：pip install -e ".[dev]"
  3. 啟動服務：在專案目錄執行 python run.py（Windows 可雙擊 start_cbdb_atlas.bat）
  4. 在瀏覽器開啟 http://127.0.0.1:8770
  5. 首次使用：在網頁點擊「更新資料」下載 CBDB；或手動將 SQLite 檔放入 data/source/cbdb.sqlite3，再執行 scripts/create_views.sh （Windows：scripts/create_views.bat）建立便利視圖。
  6. 建議（性能）：資料庫就緒後執行 python scripts/build_indexes.py，可顯著加速人物檢索與關係探索；亦可於網頁使用「構建索引」。

當前進度與已知問題

目前主要對人物、地名檢索進行了精細調整，其他區塊可能仍存在問題。 知識圖譜區塊目前僅提供篩選功能，尚未對關係進行整理——此舉是為保持資料原貌。
